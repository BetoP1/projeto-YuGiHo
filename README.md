# Projeto Yu-Gi-Oh - Jogo de Cartas

## 📌 Sobre o projeto
Este projeto consiste no desenvolvimento de um jogo inspirado no universo de Yu-Gi-Oh, com foco na implementação da lógica de jogo, interação com cartas e mecânicas básicas de batalha.

O objetivo principal é aplicar conceitos de programação através da construção de um sistema interativo baseado em regras.

---

## 🚀 Funcionalidades
- Demonstração de cartas lendárias presentes no universo de Yu-Gi-Oh
- Interface simples para interação com o usuário

---

## 🛠 Tecnologias utilizadas
- HTML
- CSS
- JavaScript

---

## 📂 Estrutura do projeto
O projeto está organizado em arquivos responsáveis por:

- Interface do usuário (HTML/CSS)
- Lógica do jogo (JavaScript)
- Manipulação de eventos e estado
